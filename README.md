# BlobMarks

BlobMarks is a bookmark manager built on top of [BlobStash](https://github.com/tsileo/blobstash).

## Features

## Roadmap

 - Web UI
 - Bookmarklet
 - Searchable (index locally)
 - CLI tools and vim plugin to retrieve stored links (when writing a blog post)
